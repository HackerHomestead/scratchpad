# Scratchpad

A retro-style notepad app with 1980s BBS aesthetics.

## Screenshot

![Scratchpad on phone](phone-screenshot.png)

## Features

- Dark theme with green-on-black terminal look
- Auto-save functionality

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3

## Build

```bash
./gradlew assembleDebug
```
